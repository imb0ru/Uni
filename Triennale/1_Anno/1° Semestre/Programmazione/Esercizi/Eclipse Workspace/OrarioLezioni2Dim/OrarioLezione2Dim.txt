Gestire l'orario delle vostre lezioni Marco Ferrara

Record Tabella:
righe, numero di righe della tabella, naturale >0
colonne, numero delle colonne della tabella, naturale >0
orario, elenco di lezioni, array di righe*colonne di Lezione

Record Lezione:
materia, array di 25 caratteri
aula, carattere, 2 oppure 4
oraInizio, reale
oraFine, reale

Lezione b = {
materia = '-'
aula = '-'
oraInizio = 0
oraFine = 0
}

Funzioni di Accesso Tabella:
1)leggereNRighe
       Input: tab, tabella di cui leggere le righe, Tabella
       Output: r, numero di righe della tabella, naturale>0

              r=campo righe di tab

2)leggereNColonne
       Input: tab, tabella di cui leggere le colonne, Tabella
       Output: c, numero di colonne della tabella, naturale>0

              c=campo colonne di tab

3)leggereLezione
       Input: tab, tabella di cui leggere la lezione, Tabella
              r, riga dove è presente la lezione, naturale>0
              c, colonna dove è presente la lezione, naturale>0
       Output: l, lezione da leggere, Lezione

              l = elemento di tab in campo orario in riga r e colonna c

4)scrivereNRighe
       Input: tab, tabella in cui scrivere le righe, Tabella
              nuovar, nuovo numero di righe della tabella, naturale>0
       Output: tab

              campo righe di tab=nuovar

5)scrivereNColonne
       Input: tab, tabella in cui scrivere le colonne, Tabella
              nuovac, nuovo numero di colonne della tabella, naturale>0
              Output: tab

              campo colonne di tab=nuovac

6)scrivereLezione
       Input: tab, tabella in cui scrivere la lezione, Tabella
              r, riga dove è presente la lezione, naturale>0
              c, colonna dove è presente la lezione, naturale>0
              l, lezione da leggere, Lezione
       Output: tab

              elemento di tab in campo orario in riga r e colonna c = l


Funzioni di Accesso Lezione:
1)leggereAula

2)leggereMateria

       Output: m, materia richiesta, array di caratteri

              m=campo materia di l

3)leggereOraInizio
       Input: l, lezione da cui leggere l'ora di inizio, Lezione
       Output: o, ora richiesta, float

              o=campo oraInizio di l

4)leggereOraFine
       Input: l, lezione da cui leggere l'ora di fine, Lezione
       Output: o, ora richiesta, float

              o=campo oraFine di l

5)scrivereAula
       Input: l, lezione in cui scrivere l'aula, Lezione
              a, aula da scrivere, carattere (2 o 4)
       Output: l

              campo aula di l=a

6)scrivereMateria
       Input: l, lezione in cui scrivere la materia, Lezione
              m, materia da scrivere, array di caratteri
       Output: l

              campo materia di l=m

7)scrivereOraInizio
       Input: l, lezione in cui scrivere l'ora di inizio, Lezione
              o, ora da scrivere, float
       Output: l

              campo oraInizio di l = o

8)scrivereOraFine
       Input: l, lezione in cui scrivere l'ora di fine, Lezione
              o, ora da scrivere, float
       Output: l

              campo oraFine di l=o
