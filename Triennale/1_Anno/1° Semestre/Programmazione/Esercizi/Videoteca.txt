Record Attore:
nome, nome dell'attore, array di caratteri max 25
cognome, cognome dell'attore, array di caratteri max 25
dataNascita, data di nascita dell'attore, array di 10 caratteri 
provinciaNascita, provincia di nascita dell'attore, array di 2 caratteri

Record Regista:
nome, nome del regista, array di caratteri max 25
cognome, cognome del regista, array di caratteri max 25
dataNascita, data di nascita del regista, array di 10 caratteri 
provinciaNascita, provincia di nascita del regista, array di 2 caratteri

Record Film:
titolo, titolo del film, array di caratteri max 50
annoProduzione, anno di produzione del film, naturale di 4 cifre
genere, genere del film, array di caratteri max 20
nazionalita, nazionalita del film, array di 3 caratteri
linguaOriginale, lingua originale del film, array di 3 caratteri
nAttoriPresenti, numero di attori presenti, naturale>0
attoriPresenti, attori che recitano nel film, array di Attore
registaFilm, regista del film, Regista

Record Disponibilita:
filmDisponibile, film disponibile in videoteca, Film
costo, prezzo del film, float
supporto, tipo di supporto del film, naturale (1=DVD, 2=Blu Ray, 3=VHS)

Record Videoteca:
nome, nome della videoteca, array di caratteri max 50
indirizzo, indirizzo della videoteca, array di caratteri max 50
filmPresenti, film presenti in videoteca, array di Disponibilita
